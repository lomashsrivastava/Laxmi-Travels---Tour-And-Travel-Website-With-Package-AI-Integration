## 📖 Description
**Laxmi Tours** is a premium travel and tourism website designed to showcase the beauty of Uttar Pradesh, with a special focus on Chitrakoot as the central hub. The platform offers a seamless experience for travelers to explore, customize, and book travel packages.

This project integrates advanced UI/UX design with functional features like a custom package builder and an AI-powered recommendation engine (simulated) to help users plan their perfect trip.

## ✨ Key Features
- **Premium UI Design**: Built with a modern, dark-themed aesthetic using glassmorphism effects, smooth gradients, and GSAP animations.
- **Dynamic Package Builder**: Users can customize their trip by selecting duration, hotel type, and vehicle preferences.
- **AI-Powered Recommendations**: An intelligent system that suggests the best travel packages based on user inputs.
- **Interactive Cart System**: A fully functional cart to manage selected packages and bookings.
- **Responsive Layout**: Optimized for all devices, ensuring a great experience on desktops, tablets, and mobile phones.
- **Visual Effects**: Includes particle animations, 3D card tilts, and scroll-triggered reveals for an immersive feel.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure and content.
- **CSS3**: Custom styling with CSS variables, Flexbox, Grid, and animations.
- **JavaScript (ES6+)**: Core logic for the package builder, cart management, and UI interactions.
- **GSAP (GreenSock Animation Platform)**: High-performance animations for scroll triggers and element reveals.
- **FontAwesome**: Scalable vector icons for a polished look.
- **Google Fonts**: 'Exo 2' and 'Orbitron' for modern and futuristic typography.
